## Golang
### Gogradle
#### build
* run following command in go main directory  
```
./gradlew build
```
or
```
gradlew.bat build
```
* execute file in gogradle directory(in linux)
* file name is combination of os/project_name(such as drawin_amd64_lobby)  
```
.gogradle/{OS}_{PACKAGENAME}
```
